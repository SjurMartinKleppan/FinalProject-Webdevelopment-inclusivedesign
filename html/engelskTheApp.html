How the app works

To use the app, the user must download the app on their mobile phone of tablet. It is available on all types of mobile digital devices.
The user must register, this allows the app to store user information as well as provides personalizes feedback.
The user ig given the opportunity to choose which areast the user wants to reduce their emission. The app will then save the information and create an overview
over how much consumption the user has over different periods (one day, one week, a month and a year) and give feedback on how the user can
reduce personal and houshold emission and save moeny on electricity, food, transportation and water.
The app is connected to various seonsors in, for example the power cabinet that sends infom to the app.
The app has extra gadgets that the user can purchase for example a tracker/sensor for the garbagecan which sends information to the app everytime the user throws away a bag of trash.
